# G810 Led GUI for Linux

Simple GUI on top of [g810-led](https://github.com/MatMoul/g810-led).

## Building for development

```sh
git clone https://github.com/viniciusbo/g810-led-gui.git
cd g810-led-gui
yarn # or npm install
yarn electron-dev
```
